# Voyage Analytics - MLOps Travel System

## Project Overview

This project focuses on building machine learning models for travel analytics and deploying them using MLOps practices.

Datasets used:

* Flights dataset
* Users dataset
* Hotels dataset

The system includes:

* Flight price prediction (Regression)
* Gender classification (Classification)
* Travel recommendation system
* API deployment
* Containerization with Docker
* Kubernetes deployment
* MLFlow experiment tracking
* Streamlit dashboard
